# Film Scanning Stand

A modular DIY film scanning stand for digitizing 35mm and medium format negatives using a mirrorless camera.

The system uses aluminium extrusions, a linear rail, leveling parts and a camera carrier. It is designed for repeatable alignment, precise camera height adjustment, and high-resolution scans of larger formats such as 6x7 and 6x12. 

## Features

- Adjustable camera height on linear rail
- Aluminium extrusion frame
- Mirrorless camera support
- Compatible with 35mm, 6x7 and 6x12 film
- Custom 3D-printed film holders
- Designed for stitched high-resolution scans
- Modular and repairable construction

## Current Status

Prototype / work in progress.
Stand is working, automation with steppers, etc. in progress
